NetDrop

About NetDrop

NetDrop is a lightweight, secure, and user-friendly tool for sharing files over a local network. Built with Flask and SocketIO, it allows users to upload, download, and manage files effortlessly through a modern web interface. With features like QR code access, real-time file updates, and password-protected authentication, NetDrop makes file sharing fast and intuitive.

How It Works

NetDrop operates as a local web server, enabling file sharing within your network. Here's how it functions:

  • Start the Server: Run the Python script to launch the NetDrop server on your machine.
  • Access via Browser: Open the provided URL (e.g., http://your-ip:5000) or scan the QR code to access the web interface.
  • Authenticate: Enter the default OTP password (2812) to log in. You can modify this in the code for added security.
  • Upload Files: Select and upload files through the interface. Files are stored in the Uploads folder.
  • Manage Files: View, download, or delete files. Real-time updates ensure the file list stays current.
  • Share Access: Others on the same network can access the interface using the URL or QR code, provided they have the password.

Installation

To set up NetDrop, follow these steps:

  1. Clone the Repository:
    git clone https://github.com/Shuuubhraj/NetDrop.git
    cd NetDrop
  2. Install Dependencies: Run the following command to install all required Python packages:
    pip install flask flask-socketio pillow qrcode
  3. Run the Tool:
    python netdrop.py
  4. Access NetDrop: Open your browser and navigate to the URL displayed in the terminal (e.g., http://192.168.x.x:5000), or scan the QR code.

System Requirements

  • Operating System: Windows, macOS, or Linux
  • Python: Python 3.6 or higher
  • Dependencies: Flask, Flask-SocketIO, Pillow, qrcode (installed via pip)
  • Network: Local network connectivity for file sharing
  • Browser: Any modern web browser (Chrome, Firefox, Safari, etc.)
  • Storage: Sufficient disk space for uploaded files

Important Notes

  • Security: The default password is 2812. Change it in the netdrop.py file (OTP_PASSWORD variable) for production use.
  • Network Access: Ensure all devices are on the same local network. Firewall settings may need adjustment to allow port 5000.
  • File Storage: Files are stored in the Uploads folder. Monitor disk space for large uploads.
  • Performance: Large file transfers may be limited by network speed and device performance.
  • QR Code: Use the QR code feature to quickly share the access URL with other devices.
  • Real-Time Updates: File uploads and deletions are reflected instantly across all connected clients via SocketIO.
